# 2inxs

Mostly C++ and Python for single board computers. Cameras, video pipelines, small services that run at home. Some Go and TypeScript when a web UI is needed.

Computer vision side: worked with YOLO, trained models, cut and prepared datasets for them.

## Projects

| Project | What it does | Stack |
|---|---|---|
| [**lanboard**](https://github.com/2inxs/lanboard) | Whiteboard for the local network. Canvas, tables with formulas, plots, a physics sandbox and encrypted chat. Works with no internet at all | Node.js · TypeScript · WebSocket · libsodium |
| [**night_watch**](https://github.com/2inxs/night_watch) | Stitches two USB cameras into a panorama and tracks whatever moves. H.265 recording, motion highlights | Python · OpenCV · GStreamer |
| [**backup_storage**](https://github.com/2inxs/backup_storage) | GUI for dumping SD cards and restoring them back. Ships as an AppImage | Python · PyQt6 |
| [**widgets**](https://github.com/2inxs/widgets) | System widget panel for Ubuntu Budgie. CPU, RAM, disks, sensors, plus a custom Plymouth boot animation | Python · GTK3 |
| [**for_home**](https://github.com/2inxs/for_home) | Wireguard hub-and-spoke that joins two apartments through a VPS. Scripts cover everything from flashing the SD card to health checks | Bash · WireGuard |
| [**rtsp_tunnel**](https://github.com/2inxs/rtsp_tunnel) | Small RTSP service for SBCs, one command deploy | C++ · GStreamer |
| [**disk_visual**](https://github.com/2inxs/disk_visual) | Cyberpunk style SSD insertion animation, plain HTML with zero dependencies | HTML5/CSS/JS |

The rest is private, drones and work things.

Пишу в основном под одноплатники: камеры, видеопайплайны, мелкие сервисы для дома. C++ и Python, местами Go и TypeScript когда нужен веб-интерфейс.

По части компьютерного зрения работал с YOLO: учил модели, резал и готовил датасеты.

- [**lanboard**](https://github.com/2inxs/lanboard) - доска для локальной сети. Холст, таблицы с формулами, графики, физическая песочница и шифрованный чат. Интернет не нужен.
- [**night_watch**](https://github.com/2inxs/night_watch) - склеивает две USB-камеры в панораму и следит за тем что движется. Запись в H.265, хайлайты по движению.
- [**backup_storage**](https://github.com/2inxs/backup_storage) - GUI для дампов SD-карт и восстановления обратно. Собирается в AppImage.
- [**widgets**](https://github.com/2inxs/widgets) - панель системных виджетов для Ubuntu Budgie.
- [**for_home**](https://github.com/2inxs/for_home) - wireguard hub-and-spoke, объединяет две квартиры через VPS. Скрипты от прошивки SD до health-check.
- [**rtsp_tunnel**](https://github.com/2inxs/rtsp_tunnel) - маленький RTSP-сервис для одноплатников, деплой одной командой.
- [**disk_visual**](https://github.com/2inxs/disk_visual) - анимация вставки SSD в стиле киберпанка, чистый HTML без зависимостей.

Остальное приватное, там дроны и рабочее.
